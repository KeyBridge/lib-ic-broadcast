CanadaStation


Attribute

Type/UOM

Description

stationType

eCanadaStationType

(String)

The CanadaStation is a consolidated / hybrid record that incorporates fields from the AM, FM, TV and SDAR station descriptions. This abstract object is not used directly, but instead is extended by the AM, FM, TV and SDAR station subtype implementations.

Channel

int

The transmitting CHANNEL; The channel value varies by Station Type and is context dependent.

Latitude

double

N.Latitude used when BORDER last calculated

Longitude

double

W.Longitude used when BORDER last calculated

bcMode

Character

Broadcasting Mode; [S: Stereo, M: Mono, P: Second Audio Channels or B: Both]

border

double

Closest distance to Canada / U.S. Border (km)

brdrLat

String

The latitude coordinates of the closest distance to the US-Canada border.

brdrLong

String

The longitude coordinates of the closest distance to the US-Canada border.

certNumb

String

Broadcasting Certificate Number. Format is "FANNNN"

City

String

The station City name.

stationClass

eCanadaStationType

The (enumerated) Canadian class code of this Station; For Canadian Stations: [A, A1, B, C, C1, LP, VLP] For Non-Canadian Stations: [A, B, B1, C, C1, C2, D]

decNumber

Int

Canadian Radio-television and Telecommunications Commission (CRTC) Decision Number. Format is "YYNNNN"

docFile

int

Industry Canada document file number

frequency

double

Broadcast frequency in MHz.

(NOTE: AM Station frequencies are originally recorded in kHz but are automatically converted to MHz upon import into the ca_station table.

network

String

The broadcast Network (CBCE, CBCF, IND, INDE, INDF, BCMH, or none)

okDump

dateTime

Last date of record update for the "Consultants" database export dump

province

String

The station province (Canada) or state (US) is located.

ssCode

String

Short spacing code; "*O#aa"

stCreat

dateTime

Date the station was first entered in the database

stMod

dateTime

Date the station record was last modified in the database

unattended

Character

Unattended operation code (Y, N)

landDistanceCanada

Double

Closest distance to Canada land edge

landDistanceFrenchNewfoundland

Double

Closest distance to French Land edge near Newfoundland

landDistanceUSA

Double

Closest distance to USA land edge







Element

Type/UOM

Description

canadaStationPK

canadaStation

The (Enumerated) station type discriminator. This column is used to

* identify the type of station in the database that this record represents.

* <p/>

* Supported valued are: [AM, FM, SDAR, TV].

antennaList

List of Antennas

List of Antenna objects. In the Canada database each antenna record describes a different polarization for the same physical antenna.

feedSignal

FeedSignal

Feed Signal table reference. This defines the source of a feed signal for a TV service.

tsid

Tsid

A 16-bit packet contained within the Extended Data Services (XDS) of EIA-608B.

comment

Comment

Comment table reference containing user information.
